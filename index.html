My Projects

algebra.ai

algebra.ai is a web application designed to make learning algebra accessible and interactive. Built using Django and a custom ORM layer, it features an adaptive practice test generator that creates personalized questions based on user input. The platform also includes:

  • Comprehensive lessons to build foundational knowledge.
  • An adaptive practice system to challenge and engage users.
  • Self-assessment tools that provide instant feedback.
  • A sleek and responsive design for seamless learning across devices.

This project showcases my dedication to creating tools that blend educational innovation with cutting-edge technology.
